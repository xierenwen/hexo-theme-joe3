# hexo-theme-joe3

Hexo port of [Halo Theme Joe3.0](https://github.com/jiewenhuang/halo-theme-joe3.0) (EJS hybrid rewrite).

## Install (git submodule)

~~~bash
git submodule add https://github.com/xierenwen/hexo-theme-joe3.git themes/joe3
~~~

In blog `_config.yml`:

~~~yaml
theme: joe3
~~~

Site-specific overrides go in blog root `_config.joe3.yml` (Hexo merges over theme defaults).
